A single-page premium portfolio. Plain HTML / CSS / JS, no build step.

## Folder contents
- index.html               The site
- assets/kinshuk-hero.jpg   TOP image (hero profile card)
- assets/kinshuk-about.jpg  BOTTOM image (About section)
- decks/                    Put your 4 strategy-deck PDFs here (see decks/README.txt)

the About image displays a touch wider. Then redeploy (see below).
